Introduction to Computer Programming

Core 3 Credits Year 1, Autumn Foundation

Course Information

Description

This is intended to be a first course in computer programming. In this course, students will study how to write computer programs in the Python language to solve simple computing problems.

Learning Outcomes

  • Understand fundamental programming concepts
  • Write Python programs to solve problems
  • Apply algorithmic thinking to real-world scenarios

Prerequisites

None

Course Category Core Course
Credit Value 3 credits
